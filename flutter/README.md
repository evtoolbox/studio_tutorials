Пример интеграции flutter в мобильное приложение. Версия EV Toolbox > 3.6.0.
В примере представлена модель логотипа компании, которую можно вращать касанием и реализованный на flutter пользовательский интерфейс, состоящий из трех элементов: иконки показа/скрытия модели, кнопки проигрывания аудио и надписи, отображающей название узла, на котором произошло касание.
